# Proyecto de Juegos - TicTacToe y Conecta 4

Este proyecto permite jugar a dos juegos clásicos: TicTacToe y Conecta 4. A continuación se detallan las instrucciones para ejecutar el proyecto y jugar a los juegos.

## Requisitos

- **Java JDK 8 o superior**: Asegúrate de tener instalado el Java Development Kit (JDK). Puedes descargarlo desde [aquí](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html) o utilizar un administrador de paquetes según tu sistema operativo.
- **IDE**: Se recomienda usar un IDE como IntelliJ IDEA, Eclipse o NetBeans para facilitar la ejecución y edición del código.

## Ejecución del Proyecto

1. Clona este repositorio o descarga los archivos del proyecto.
2. Abre el proyecto en tu IDE.
3. Asegúrate de que el archivo `JuegoControlador.java` sea el archivo principal.
4. Compila y ejecuta el programa.

## Seleccionar y Jugar los Juegos

### TicTacToe

1. Cuando ejecutes el programa, se te presentará un menú.
2. Selecciona la opción `1` para jugar TicTacToe.
3. Sigue las instrucciones en pantalla para hacer tus movimientos. El juego terminará cuando un jugador gane o haya un empate.
4. Al finalizar, se te preguntará si deseas jugar de nuevo.

### Conecta 4

1. Después de seleccionar el juego, si eliges `2`, iniciarás el juego de Conecta 4.
2. Al igual que en TicTacToe, sigue las instrucciones en pantalla para realizar tus movimientos. El objetivo es alinear cuatro fichas en horizontal, vertical o diagonal.
3. Al finalizar, se te preguntará si deseas jugar de nuevo.

## Detalles Relevantes

- **Salida del programa**: Puedes salir del programa en cualquier momento seleccionando la opción `3` en el menú principal.
- **Manejo de entradas**: Asegúrate de ingresar opciones válidas (1, 2 o 3) cuando se te solicite, de lo contrario, se te pedirá que lo intentes nuevamente.
- **Reiniciar el juego**: Después de terminar un juego, puedes optar por jugar de nuevo o salir del programa.

¡Diviértete jugando a TicTacToe y Conecta 4!
